# 🙇 Personal Portfolio

My personal portfolio site, created as part of an assessment in Term 1 of the Holmesglen Institute's Diploma of Web Development course.

## ⚙️ Skills

**SASS**

- My first project using SCSS
- SASS variables, nesting, modules, and inheritance

**React**

- Also my first project using React
- Components to segment aspects of the website
- useState and useEffect to allow for a dynamic re-rendering of website components
- React Router (and hashlinks) to enable re-rendering page links
